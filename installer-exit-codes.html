Installer Exit codes

All exit codes for installer

  • 0 - Installation completed successfully
  • 1 - Setup failed to initialize
  • 2 - Installation cancelled before starting
  • 3 - Internal error occurred during setup preparation
  • 4 - Installation failed during execution
  • 5 - Installation cancelled during process
  • 6 - Setup terminated unexpectedly
Any non-zero exit code indicates that the installation did not complete successfully.
